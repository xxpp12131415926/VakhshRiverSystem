# Pamir Snowmelt Runoff Warning System (Pamir-SRWS)
**帕米尔高原融雪径流预警系统 (基于 GEE Python API 的物理引擎)**

## 🏔️ 项目背景
在极端缺资料流域（Ungauged Basins），传统的水文验证和地面监测手段面临巨大挑战。本项目聚焦于横跨多国的**帕米尔高原完整自然地理单元**，旨在解决高寒山区无实测资料条件下的径流预测痛点。

系统将研究目标从单纯的积雪面积提取，全面升级为**融雪径流发生概率的连续空间映射**。通过 Google Earth Engine (GEE) Python API，无缝调度云端算力，为下游气象预警和大屏展示系统提供高度解耦的后端支持。

## ✨ 核心科学特性
* **高密度自然地理边界融合**
  摒弃了粗糙的几何裁切方式。系统将“Pamir alpine desert and tundra”生态区与 3000 米以上高程数据进行空间并集，提取出贴合真实山脊线与生态过渡带的复杂多边形，完美覆盖帕米尔高原的跨国自然边界。
* **Sigmoid 软阈值物理相变模型**
  针对微波（SAR）信号的后向散射特征，引入基于 Sigmoid 函数的软阈值概率模型，替代传统的硬阈值切割，更真实地反映积雪从干雪到湿雪的连续物理相变过程。
* **大尺度动态地形热力学校正**
  结合 DEM 阴阳坡数据，动态计算不同热力条件下的平均湿雪海拔，并赋予防崩溃极值，有效消除雷达阴影和复杂地形对雪线判读的干扰。
* **无资料地区 AHP 概率映射**
  在缺乏地面验证数据的情况下，采用层次分析法 (AHP) 融合距河流距离、坡度、坡向等关键地形水系因子，直接输出极高产流风险区。
